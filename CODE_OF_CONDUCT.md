# Code of Conduct

Certior is committed to a respectful, professional, and inclusive
environment for everyone who participates in this project - as
contributors, reviewers, maintainers, or users.

## Expectations

When taking part in this project (issues, pull requests, discussions,
or any other community space), please:

- Treat others with respect and good faith.
- Assume positive intent and ask clarifying questions before reacting.
- Keep discussion focused on the work and on ideas, not on people.
- Welcome contributors of all backgrounds and experience levels.
- Acknowledge contributions and credit work appropriately.

Conduct that is disrespectful, demeaning, or that targets individuals
or groups will not be tolerated in this project's spaces.

## Reporting

If you observe behaviour that violates this code of conduct, please
report it in confidence at
[hello@certior.io](mailto:hello@certior.io).

Reports will be reviewed promptly and confidentially. The maintainer
may, at their discretion, take any action they consider appropriate,
including warnings, temporary or permanent removal from project
spaces, or referral to the relevant platform.

## Scope

This code of conduct applies to all project-controlled spaces,
including the GitHub repository, issue tracker, pull requests,
documentation site, and any official communication channels.
