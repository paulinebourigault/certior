# `docs/` - Certior documentation

This directory holds both the **public** Certior documentation (published to [docs.certior.io](https://docs.certior.io) via Mintlify) and the **internal** working documents (`internal/`, not published).

## Published docs (Mintlify)

Edit any of these `.md` / `.mdx` files and push to `main`; Mintlify rebuilds in ~60 seconds. The navigation tree is in [`docs.json`](docs.json).

| File | Tab in Mintlify nav | Audience |
|---|---|---|
| [`introduction.mdx`](introduction.mdx) | Documentation → Get started | First-time landing page |
| [`quickstart.md`](quickstart.md) | Documentation → Get started | Universal 5-minute install + first verified call |
| [`guides/openai.md`](guides/openai.md) | Documentation → Guides | OpenAI tool-calling integration recipe |
| [`bring-your-own-framework.md`](bring-your-own-framework.md) | Documentation → Integrate | LangChain / CrewAI / MCP recipes |
| [`openclaw-defenses.md`](openclaw-defenses.md) | Documentation → Integrate | OpenClaw threat-class → gate mapping |
| [`INTEGRATIONS.md`](INTEGRATIONS.md) | Documentation → Integrate | GitHub Action, Trust Badge |
| [`api-contract.md`](api-contract.md) | Documentation → Operate | REST + WebSocket API reference |
| [`lean-binary.md`](lean-binary.md) | Documentation → Operate | Building and shipping the Lean policy kernel |
| [`certior-trust-package.md`](certior-trust-package.md) | Documentation → Understand | Auditor-facing assurance docs |
| [`openclaw-skill-audit.md`](openclaw-skill-audit.md) | - | `certior-skill-audit` CLI reference |

## Surfaces that link in from here

- **Landing site** ([`landing/`](landing/)) - `certior.io`. See [`landing/README.md`](landing/README.md) for the deploy recipe.

## Static assets

- [`assets/`](assets/) - architecture diagrams referenced from the published docs (e.g. `architecture-hero.svg`).
- [`favicon.png`](favicon.png) - the Mintlify tab icon (Certior org logo).
- The HIPAA-preset compliance export is generated by [`examples/03_hipaa_compliance.py`](../examples/03_hipaa_compliance.py); run it to see what an audit export looks like.

## Local docs preview

```bash
# Landing page (Tailwind CDN + static HTML)
cd docs/landing && python -m http.server 8080
# → http://localhost:8080

# Mintlify preview (requires Mintlify CLI)
cd docs && mintlify dev
# → http://localhost:3000
```
